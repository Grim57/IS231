# Вопросы к зачету 

Для получения зачета к МДК 10.01 "Обработка отраслевой информации"   
следует ответить на 3 вопроса из набора:
```
В-1 Что такое SMM в современном маркетинге?
В-2 Какова главная цель SMM в отличие от контекстной рекламы?
В-3 Какой главный принцип следует использовать при выборе социальных платформ для бизнеса?
В-4 Какой показатель лучше всего отражает вовлеченность аудитории?
В-5 Что означает «единый стиль» визуального контента бренда?
B-6 Почему для бренда важно отвечать на комментарии под постами?
В-7 Что включает в себя «ведение сообществ»?
В-8 Почему микро-инфлюенсеры эффективны для продвижения?
В-9 Почему вертикальное видео стало трендом?
B-10 В чем главное преимущество платформы «Дзен» для SMM-специалиста?
B-11 Ключевое ограничение (минус) ведения бизнес-страницы во «ВКонтакте» — это?
В-12 С какого поста рекомендуется начать ведение аккаунта в соцсетях?
В-13 Главное конкурентное преимущество платформы YouTube для бизнеса — это?
B-14 Какая организация уполномочена блокировать ресурсы и контент на территории РФ за нарушение законодательства?
В-15 Что является ключевым ограничением (минусом) платформы TikTok для бизнеса?
В-16 Для чего в первую очередь бренду стоит использовать YouTube?
В-17 Чем регулируется защита персональных данных в РФ?
В-18 Владелец цветочного магазина поставил цель: «Увеличить продажи букетов к 8 Марта». Какой критерий SMART цели нарушен?
В-19 Какой показатель лучше всего подходит для измерения цели «Повысить узнаваемость бренда»?
В-20 Что измеряет и какова формула коэффициента вовлеченности (ER)?
В-21 Какой критерий SMART нарушает цель: увеличить количество подписчиков в группе ВКонтакте с 1000 до 5000 за 1 неделю?
В-22 Что такое и для чего используются UTM-метки?
В-23 Что такое целевая аудитория (ЦА)?
В-24 Косвенный конкурент для кофейни — это?
В-25 Главная цель определения ЦА для SMM-специалиста — это?
В-26 Завершите утверждение: «Сильные и слабые стороны в SWOT-анализе — это... факторы»
В-27 Уникальное Торговое Предложение (УТП) — это?
```
Ответы должны быть даны без использования телефона либо компьютера, на листе бумаги (возьмите, пожалуйста, с собой ручки)  
Вам предоставлен невероятный шанс самостоятельно подготовиться к зачету!  

Удачи и отличных ответов на зачете!
