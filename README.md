## Разработать структуру проекта для блога

Создайте базовую структуру проекта для блога, реализуете маршруты, модель и контроллер.   
Это станет отличной основой для создания более сложных приложений на PHP.  

Работа с git
```
- Войдите в свой аккаунт на github.com и сделайте fork на репу https://github.com/Coopteh/IS231
- Снимите галочку с "Copy the main branch only"
```
